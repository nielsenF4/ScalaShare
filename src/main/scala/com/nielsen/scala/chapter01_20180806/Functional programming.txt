函数式编程：
函数式编程的重要概念：
1.副作用
   var x = 1
   def XplusY_V1(y: Int) = x + y
   def XplusY_V2(y: Int) = {x += y; x}
   XplusY_V1(2)  //此时x还是1
   XplusY_V2(2)  //此时x是3
 XplusY_V1对x没有产生副作用是一个纯函数
2.引用透明
   对于相同的输入，总是得到相同的输出
   如果f(x)的参数x和函数体都是引用透明的，那么函数f是纯函数
   违反引用透明：  var x = new StringBuilder("Hello")
               var y = x.append(" World!")
               var z = x.append(" World!")
   append是违反引用透明的，因为每次返回的结果都不一样
3.不变性
   为了获得引用透明性，任何值都不能变化
4.函数是一等公民
   一切都是计算，函数式编程中只有表达式，变量、函数都是表达式
	函数是第一等公民：
	1).把函数作为实参传递给另外一个函数
	2).把函数作为返回值
	3).把函数赋值给变量
	4).把函数存储在数据结构里

 函数式编程的优点：
1. 生产效率高
2. 易于推理
3. 并行编程
4. 多核计算、云计算
